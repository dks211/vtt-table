# Maps

Drop your battle map images here — Inkarnate exports, screenshots, anything the console can import (PNG, JPG, WebP).

The console doesn't read this folder automatically; it's just a tidy place to keep your maps so they live alongside the campaign. To use one, open the console, switch to the **YOUR MAP** tab, and drag the image onto the canvas (or use **IMPORT MAP IMAGE**).

Suggested naming so future-you can find things:

```
casino-level-1.png
casino-level-2-vault.png
refugee-city-streets.jpg
```
